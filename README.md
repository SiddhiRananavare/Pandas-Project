# Pandas-Project
Pandas-based data cleaning and exploratory data analysisEmployee Data Analysis using Pandas

📌 Project Overview

-This project focuses on analyzing employee data using Pandas to extract meaningful insights. It demonstrates how raw data can be cleaned, processed, and analyzed to support data-driven decision-making.

🛠 Tools & Technologies
-Python
-Pandas
-NumPy

📊 Steps Performed
-Data Collection & DataFrame Creation.
-Data Cleaning (handling structure & formatting).
-Exploratory Data Analysis (EDA).
-Data Filtering & Sorting.
-GroupBy & Aggregation Operations.
-Basic Statistical Analysis.

🎯 Result
-Identified department-wise salary trends.
-Analyzed employee distribution across departments.
-Extracted meaningful insights from structured data.
-Built a strong foundation for real-world data analysis tasks.
